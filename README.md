# Ücretsiz Udemy Kuponları
Yazılan script sadece ilk sayfadaki verileri alıyor. Sayfa sayısını arttırmak için aşağıdaki kod bloğunu şu şekilde değiştirmeniz yeterli olacaktır.

```
i = 1
while i <= 2:
  ...
```
Değiştirilmiş hali:
```
i = 1
while i <= 10:
    ...
```
